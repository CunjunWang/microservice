# microservice
- 基于Spring Cloud部署微服务

## 技术栈
- 服务的注册与发现：Spring Cloud Eureka (计划改用Consul)
- 反向代理：Nginx
- 负载均衡：Ribbon
- 服务间通信：Openfeign
- 消息队列：RabbitMQ (计划改用 ActiveMQ)
- 服务网关：Spring Cloud Zuul
- 熔断机制： Hystrix
- 分布式跟踪：Spring Cloud Zipkin + Spring Cloud Sleuth
- 数据库：MySQL
- 缓存：Redis
- 部署：Docker + Rancher
